City of Los Angeles

Desktop Application / Responsive Design

The City of Los Angeles' Human Resource Management System is an internal application that serves as a central repository of employee information. The large majority of the site consists of input fields, so my focus was to create easily reusable styles for these inputs. I implemented a six-column grid system for accommodating input fields and their labels. The site is responsive down 960 pixels wide to account for smaller browser sizes.
