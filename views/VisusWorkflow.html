Visus Workflow

Desktop Applilcation / Logo Design / SASS

Visus Workflow is a workflow process management tool. Users can create workflows for projects and assign tasks to members of their team. The application includes drag-and-drop functionality, and the ability to create workflow templates and manage user roles. I created a clean and minimalist style with colorful accents to show task status and element functionality. The logo is representative of moving from one task in a workflow to another.
